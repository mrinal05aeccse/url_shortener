LangGraph integration notes

This folder contains a simple example (backend/langgraph/example.py) that demonstrates how one
might call an LLM orchestrator (LangGraph) to summarize analytics for a given alias.

The example is intentionally a placeholder. For production integration:
- Install the official LangGraph SDK if available.
- Use secure API keys via environment variables or a secrets manager.
- Implement retries and error handling for network calls.
